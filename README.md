# DM_Lab1
Интеллектуальный анализ данных. Лабораторная номер 1 «Первичное исследование и оценка качества данных»

Цель работы:
После выполнения лабораторной работы студент должен уметь:
- Описывать датасет в терминах задачи интеллектуального анализа данных
(кто заказчик, какие вопросы к данным).
- Выполнять первичный обзор данных: размер, структура, типы признаков,
базовая статистика.
- Проводить базовый аудит качества данных: пропуски, дубликаты,
подозрительные значения, выбросы.
- Строить несколько простых, но осмысленных визуализаций для
разведочного анализа.
- Оформлять результаты в виде:
• воспроизводимого Jupyter Notebook;
• краткого текстового отчёта с выводами и планом улучшения качества
данных;
• аккуратно организованного репозитория в системе контроля версий.

В качестве исходныз данных взят CSV файл с кинофильмами. Подробнее о файле можно узнать в паспорте проекта (есть в report/report_quality и в notebooks/01_data_understanding.ipynb).

Запуск производится в файлк notebooks/01_data_understanding.ipynb по порядку следования блоков кода
